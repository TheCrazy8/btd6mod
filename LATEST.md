# BTD6 Chinese Skins v1.1.0

## Features
- Enables Chinese exclusive skins in the international version of BTD6
- Includes New Year skins for:
  - **Quincy** - General Quincy skin with Chinese armor
  - **Psi** - Qinglong (Azure Dragon) Psi skin
  - **Sauda** - Catwoman Sauda skin (never released internationally!)

## Installation
1. Make sure you have BTD6 v52.0 installed
2. Install MelonLoader and BTD6 Mod Helper
3. Download `BTD6Mod.dll` from this release
4. Place it in your BTD6 `Mods` folder
5. Launch the game and the skins will be automatically unlocked!

## Notes
- Skins are automatically unlocked when you reach the main menu
- Save game compatibility is maintained - saves will work even if you remove the mod
- This mod is based on the Chinese edition of BTD6 and ports the exclusive content to international versions
